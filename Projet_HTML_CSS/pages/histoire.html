INTRODUCTION

Aujourd'hui, les programmes informatiques permettent la productions de nos sites internet ainsi que nos applications. Nous allons voir quelles sont les origines de la création de ces programmes.

LES ORIGINES

Bien avant l'apparition des ordinateurs, Ada Lovelace(1815-1852) traduisait les travaux de Charrles Babbage sur sa machine, elle a crée le premier algorythme informatique pour calculer des nombres. Ada est considérée comme la première programmeuse au monde.

LES PREMIERS ORDINATEURS MODERNES

Les premiers ordinateurs électroniques apparaissent durant les années 1940-50. Pendant cette période, Konrad Zuse conçoit le premier langage de programmation. En 1950, les premiers véritables langages informatiques voient le jour. Le Fortran (Formula Translator), créé par John Backus, devient le premier langage fonctionnel pour le calcul scientifique. Pendant la meme période, Grace Hopper développe le premier compilateur, une innovation qui permet de traduire un code en un langage compréhensible par la machine.

EXPLOSION DE LA PROGRAMMATION

Durant les années 1960, la programmation explose avec la création de langages encore utilisés aujourd’hui. Par exemple, le langage C, crée entre 1969 et 1973 par Dennis Ritchie, qui est encore présent dans de nombreux systèmes modernes. Le Pascal, inventé en 1970, est utilisé pour développer des logiciels célèbres comme la première version de Photoshop. Dans les années 1980, les langages évoluent pour répondre à de nouveaux besoins:

  • C++ (1983) introduit des concepts de programmation orientée objet.
  • Objective-C, utilisé par Apple, révolutionne les interfaces graphiques.
  • SQL (1974), encore incontournable aujourd’hui, permet de gérer des bases de données relationnelles.

L'ARRIVEE D'INTERNET

Avec l’arrivée d’internet, de nouveaux langages apparaissent pour répondre aux besoins du web.

  • Python (1991), conçu pour être simple et lisible, devient un outil essentiel pour les développeurs, notamment dans la recherche scientifique.
  • JavaScript (1995) révolutionne les pages web en permettant des interactions dynamiques et intuitives.

Ces innovations marquent le début d’une nouvelle ère, où le développement web devient présent dans la vie de tout les jours.

LA PROGRAMMATION AUJOURD'HUI

Aujourd’hui, les langages de programmation continuent de s’adapter aux évolutions technologiques, l’intelligence artificielle aux objets connectés. Le code est partout : il gère nos voitures, analyse des données et transforme l’économie mondiale.
